Refit - Responsive Portfolio & Service Website






Project Overview

Refit is a fully responsive, modern service website built with HTML, CSS, and JavaScript, designed for showcasing business services, projects, and client testimonials. Itâ€™s ideal for service-based businesses like construction, interior design, or renovation companies.

Features

Responsive Design: Optimized for desktop, tablet, and mobile.
Interactive Services Section: Clickable service cards that display smooth image transitions and expandable descriptions.
Animated Counters: Highlight key statistics with animated number counters.
Testimonials Slider: Auto-scrolling client testimonials powered by Swiper.js.
FAQ Accordion: Expandable FAQ items with smooth icon rotation and content reveal.
Contact Form: Simple contact form with inputs for name, email, phone, and message.
Mobile Navigation: Hamburger menu for mobile devices with slide-down effect.
Modern CSS Design: Uses Flexbox, Grid, transitions, hover effects, and fade-in animations.
Icons: Integrated Ionicons for visual appeal.

Technologies Used

HTML5
CSS3 (Flexbox & Grid)
JavaScript
Swiper.js for sliders
